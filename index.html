Goals

Enhance Technical skills

I intend to improve my proficiency in HTML, CSS and Javascript. This can be achieved by working on multiple projects that involve creating complex layouts, animations and interactivity.

Be adept in Frontend Frameworks and Libraries

Gain expertise in popular frontend frameworks like React, Angular, or Vue.js. This can be achieved by collaborating and contributing to projects that utilize these frameworks

Build Responsive and Accessible web Applications

Ensuring well applications are responsive and accessible to all users. Testing and validating applications on various devices and with accessibility tools, aiming for compliance with accessibility guidelines.

Improve UI/UX Design Skills

Develop a strong understanding of user experience and user interface design principles. Collaborate with design teams to learn about UI/UX best practices and apply them to frontend development.

Optimize Web Performance

Understand and implement web performance optimization techniques. Study performance bottlenecks, using tools like Lighthouse to audit performance, and implement best practices for optimization.

Gain More Experience with API integration

Improve skills in integrating and working with APIs. Work on projects that require fetching, displaying, and manipulating data from various APIs.
